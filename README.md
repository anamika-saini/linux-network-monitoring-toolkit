# 📖 Overview

The **Linux Network Monitoring Toolkit** is a collection of Bash scripts designed to simplify Linux network administration and troubleshooting.

The toolkit automates common networking tasks including monitoring network interfaces, checking active TCP/UDP connections, verifying DNS configuration, testing network connectivity, inspecting listening ports, collecting bandwidth statistics, and generating comprehensive network health reports.

This project demonstrates practical Linux system administration skills commonly required for:

- AWS Cloud Support Engineer
- Linux System Administrator
- Technical Support Engineer
- Junior DevOps Engineer
- Cloud Operations Engineer
- Site Reliability Engineer (SRE)

# ⚙️ Installation

Clone Repository

```bash
git clone git@github.com:anamika-saini/linux-network-monitoring-toolkit.git
```

Move inside project

```bash
cd linux-network-monitoring-toolkit/scripts
```

Give execute permission

```bash
chmod +x *.sh
```

Run any script

```bash
./network_status.sh
```

# 📜 Script Details

| Script | Description |
|----------|-------------|
| network_status.sh | Displays complete system and network information |
| active_connections.sh | Shows active TCP/UDP connections |
| ping_test.sh | Tests network connectivity |
| dns_lookup.sh | Performs DNS lookup using Host, Dig & Nslookup |
| port_scanner.sh | Displays listening TCP/UDP ports |
| bandwidth_monitor.sh | Shows network interface statistics |
| network_report.sh | Generates a complete network diagnostic report |
